========================================================================
    静态库：libCommon 项目概述
========================================================================

应用程序向导已为您创建了此 libCommon 库项目。

本文件概要介绍组成 libCommon 应用程序的每个文件的内容。


libCommon.vcproj
    这是使用应用程序向导生成的 VC++ 项目的主项目文件，
    其中包含生成该文件的 Visual C++ 的版本信息，以及有关使用应用程序向导选择的平台、配置和项目功能的信息。


/////////////////////////////////////////////////////////////////////////////

StdAfx.h, StdAfx.cpp
    这些文件用于生成名为 libCommon.pch 的预编译头 (PCH) 文件和名为 StdAfx.obj 的预编译类型文件。
